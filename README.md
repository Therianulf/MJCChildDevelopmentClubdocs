# MJC Child Development Club - Marketing Materials

Marketing materials for the Modesto Junior College Child Development Club. All flyers are HTML files designed to be printed as clean PDFs directly from your browser.

## Flyers

Open any flyer HTML file in your browser, then **File > Print > Save as PDF**. They're sized for 8.5x11" letter paper and will print cleanly with no extra setup.

| File | Description |
|------|-------------|
| `flyers/club-info.html` | Club recruitment flyer - who we are, what we do, when & where we meet |
| `flyers/ece-cd-flowchart.html` | 4-step pathway to becoming an ECE or Child Development major |
| `flyers/multicultural-day.html` | Event flyer for Multicultural Day (March 13, 2026) |

## Website

`website/index.html` - A single-page club website meant to be hosted online. **This is still a work in progress** and not yet ready for deployment.
